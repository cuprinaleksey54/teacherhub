# TeacherHub (frontend прототип)

Тёмный современный прототип сайта для молодых учителей: главная страница, регистрация/вход (пока без сервера), вкладки **AI помощник / Уроки / Журнал / Шаги / FAQ / Сообщество / Техподдержка**.

## Запуск

Так как это простой HTML/CSS/JS, можно открыть `index.html` в браузере, но корректнее запускать через локальный сервер:

```bash
cd /Users/user/teacherhub
python3 -m http.server 5173
```

Потом открыть в браузере:
- `http://localhost:5173`
